// 以下是注释区（其实这个双斜杠//符号打不打都行，这后面的部分程序读进去只会认为是没用的部分）

// 目标函数OF只有一条，如果分号分隔多条的话会以最后一条目标函数为准

// 变量名不要数字开头！规定最多只能是三个字符，如：x1,x2,x12,x99，多余的会被忽略
// 价值系数可以是整数、分数、小数，
// 写分数的时候不要把符号写在分母上，另外写成分数的话分子分母都得是整数
// 大括号中每一项请务必用分号;分隔，不然可能有意想不到的结果

// 决策变量有上限下限时只能写成多条：比如0<=x1<=5要写成x1>=0;x1<=5
// 单纯形法要求所有决策变量非负，出现x1<=9这种无法判断的约束时会报错
// 选择整数规划的时候，所有决策变量都会自动新增整数约束
// (如果将来会写0-1整数规划的话）所有决策变量自动约束只能取0,1
