CONSTANTS {
	M>=0;C
}

LF {
	max  : z  =  4.5-M   -245/100Mx1 + 1.35x2 -245/5M  +  2/4M +  1.235  -  x3 - M;
}
ST {
	x1 + 2x2 <= -1.35;
	4x1      <= 16;
	       4x2<=12;
	x1>=0;
	x2>=0
}

// CONSTANTS中的常量只支持一个英文字母a-zA-Z进行表示，常量可以作为系数。
// 默认常量>0（比如A相当于A>0），如果写成A<0就表示A是小于0的常量。当然A>6 A<=5 A=5这种写法也是可以的（等号的话常量会被直接和前面的系数相乘）
// 为了处理方便常量在系数中请务必写在数字后面：3M/4  2.45M  3M ，不要想不开这样写：M3 M3/4
// 一个系数请不要出现多个常量：3CM/4 3C/4M 这样写是绝对不行的
// 按理来说应该不怎么会用到常量，这里主要是为了大M法
// 目标函数LF只有一条，如果分号分隔多条的话会以最后一条目标函数为准
// 变量名不要数字开头！规定最多只能是三个字符，如：x1,x2,x12,x99，多余的会被忽略
// 价值系数可以是整数、分数、小数，
// 写分数的时候不要把符号写在分母上，另外写成分数分子分母都得是整数
// 大括号中每一项请务必用分号;分隔，不然可能有意想不到的结果
